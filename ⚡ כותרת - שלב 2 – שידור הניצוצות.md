## 🎯 שלב 2: שידור הניצוצות 🔄

המשך ישיר של _גאולה בנעימים – ההתעוררות_, שבו עוברים מהקמה והגדרה — אל **שגרה רוחנית־טכנולוגית יציבה**.  
המטרה בשלב זה: להפוך את השידור החד־פעמי למערכת חיים דינמית ומתמשכת של אור, תודעה וקהילה פעילה 💫

---

### 🌟 מטרות מרכזיות

|#|מטרה|תיאור|
|---|---|---|
|1|**בניית לוח שידורים קבוע (שבועי/חודשי)**|לקבוע ימים ושעות קבועים לשידור – כדי ליצור יציבות והמתנה בקרב הקהל.|
|2|**יצירת תבנית שידור סטנדרטית**|תבנית אחידה לפתיח, תוכן, אינטראקציה וסיום – שתשמש לכל פרק בלייב.|
|3|**פיתוח מערכת התראות לקהילה**|מערכת שתשלח הודעות לפני ואחרי שידור (Mail, Telegram, WhatsApp).|
|4|**אינטגרציה עם רשתות חברתיות נוספות**|חיבור אוטומטי לשידור מקביל בפייסבוק, טוויץ’, אינסטגרם וטיקטוק.|

|🌐 פיתוח מערכת ניהול קהילה|תיעוד משתמשים, תגובות, אינטראקציות – דרך API או Google Sheet|🔄 בתהליך|
|🧠 כתיבת מדריך “איך לשדר ניצוץ”|מסמך הדרכה לכל מי שרוצה להצטרף לצוות שידור או לפתוח ערוץ אור משלו|🟢 נבנה כעת|

### 🌐 אינטגרציה מוצעת (GitHub + רשתות)

- **GitHub Actions:** ימשוך אוטומטית את לוח השידורים מה־README או מקובץ YAML ייעודי.
    
- **Zapier / n8n / Make:** ישמשו כגשר בין GitHub → YouTube → Telegram → Facebook.
    
- **Google Sheets API:** יתעד נתוני צפייה, תגובות, ותאריכי שידורים באופן רציף.
    
- **RSS + Embed:** ליצור עמוד “Live Feed” באתר AnLoMinus שיראה את כל השידורים האחרונים.
